Cancel Order Magento Extension

The Cancel Order extension for Magento 2 allows customers to cancel their orders directly from the customer dashboard, providing a simple, configurable solution to manage cancellations. Key features include customizable cancellation reasons, admin notifications, and an optional time-limited cancellation window. Up until Magento 2.4.7, there is no native option for customers to cancel orders from the frontend. Our extension meet this requirement for stores running on previous versions. Also GraphQL integrated.
